Требования:
1. Данный модуль интеграции с kaznachey разработан для Joomla версии 1.5.x;
2. Модуль разработан для компонента VirtueMart, т.е. перед установкой модуля установите этот компонент.

Перед установкой модуля необходимо зарегистрироваться на сайте kaznachey.ua (раздел "ПОДКЛЮЧЕНИЕ"). 
После подписания договора быдут выданы: 
-идентификатор мерчанта; 
-секретный ключ мерчанта; 
Переданные данные будут использованы при настройки модуля
	
a) Success URL - страница, на которую будет перенаправлен плательщик после оплаты;
b) Fail URL - страница, на которую будет перенаправлен плательщик при отказе от оплаты;
c) Result URL (заполняется после установки модуля) - на этот адрес будет отправляться уведомление о состоянии платежа;
d) Секретный ключ - произвольный набор символов, используется для подписи сообщений при переадресации;	
e) Скрытый ключ - произвольный набор символов, используется для подписи скрытых сообщений на Result URL;	
f) Ключ API - произвольный набор символов, используется в случае дальнейшего расширения функций модуля разработчиками проекта.

Порядок установки модуля:
1. Скопируйте папку Administrator в корневую директорию сайта;
2. Скрипт, содержащийся в файле kaznachey_result.php обрабатывает сообщения о состоянии платежа. Скопируйте его в любое место на вашем сайте, так чтобы он был доступен извне. Адрес этого скрипта должен быть указан в поле Result URL в панели управления магазином на сайте kaznachey.com;
2. Зайдите в панель управления вашего сайта;
3. Откройте управление компонентом VirtueMart (меню Components/VirtueMart);
4. Выберите пункт меню Store/Add payment methods;
5. Заполните поля:
 - Active: отметьте галочкой, если хотите, чтобы этот способ оплаты отобразился на сайте;
 - Payment Method Name: Название способа оплаты, которое будут видеть ваши клиенты (Например: kaznachey);
 - Code: Код способа оплаты. Укажите в этом поле значение "kaznachey";
 - Payment class name: Название PHP-класса, который отвечает за настройку этого способа оплаты. Укажите в этом поле значение "ps_kaznachey";
 - Payment method type: Тип способа оплаты. Выберите значение "PayPal (or related)";
6. Перейдите на вкладку "Configuration";
7. Заполните поля:
 - Merchant ID: Идентификатор вашего магазина в системе kaznachey, значение указано в панели управления магазином на сайте kaznachey.com;
 - Secret Key: Секретный ключ, значение указано в панели управления магазином на сайте kaznachey.com;
 - Статус заказа для подтвержденных платежей: Должно быть выбрано значение "Confirmed";
 - Статус заказа для отмененных платежей: Если при отказе оплаты нужно отменять заказ, то выберите "Canceled". Если хотите дать возможность после отказа повторно переходить к оплате, выберите "Pending";
 - Payment Extra Info: Вставьте в это поле содержимое следующий код:
 <?php kaznachey(); ?>
 
8. Сохраните изменения.
